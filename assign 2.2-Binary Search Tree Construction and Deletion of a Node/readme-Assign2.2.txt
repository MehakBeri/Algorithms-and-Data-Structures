Source code is in java. This program constructs a binary search tree and then deletes a number in the binary search tree.

To run the file, please follow the steps below:
1. Open command line
2. Compile file by typing "javac Assign22.java"
3. Run the program by typing "java Assign22"
4. You shall see the inorder traversal of the binary search tree created in the program.
5. The program asks for a user input and deletes the number entered by the user from the binary search tree.
6. After deletion, inorder traversal of the binary search tree is done again.
7. The program will allow you to delete a number from the binary search tree if you keep typing 'y'.
8. Enter any other character to exit from the program.
